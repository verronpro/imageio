= ImageIO SVG

The `imageio-svg` module provides a plugin for the Java ImageIO API to read Scalable Vector Graphics (SVG) files.
It uses Apache Batik for high-quality rendering of SVG content into `BufferedImage`.

== Key Features

* **ImageIO Integration**: Automatically registered as a provider for SVG files.
* **High Quality**: Leverages Apache Batik for accurate vector rendering.
* **Simple API**: Use `ImageIO.read()` to load SVG files just like any other image format.

== Quick Start

[source,java]
----
BufferedImage image = ImageIO.read(new File("image.svg"));
----

== Maven Coordinates

[source,xml]
----
<dependency>
    <groupId>pro.verron.imageio</groupId>
    <artifactId>imageio-svg</artifactId>
    <version>${project.version}</version>
</dependency>
----
