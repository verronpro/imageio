= ImageIO EMF

The `imageio-emf` module provides a plugin for the Java ImageIO API to read Enhanced Metafile (EMF) files.
It allows for high-quality rendering of EMF content into `BufferedImage` using the standard ImageIO interface.

== Key Features

* **ImageIO Integration**: Automatically registered as a provider for EMF files.
* **Vector Quality**: Preserves the vector quality of metafile content during rendering.
* **Simple API**: Seamlessly integrates with `ImageIO.read()`.

== Quick Start

[source,java]
----
BufferedImage image = ImageIO.read(new File("image.emf"));
----

== Maven Coordinates

[source,xml]
----
<dependency>
    <groupId>pro.verron.imageio</groupId>
    <artifactId>imageio-emf</artifactId>
    <version>${project.version}</version>
</dependency>
----
