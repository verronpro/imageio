= ImageIO WMF

The `imageio-wmf` module provides a plugin for the Java ImageIO API to read
Windows Metafile (WMF) files.
It allows for high-quality rendering of WMF content into `BufferedImage`
using the standard ImageIO interface.

== Key Features

* **ImageIO Integration**: Automatically registered as a provider for WMF files.
* **Vector Quality**: Preserves the vector quality of metafile content during rendering.
* **Simple API**: Seamlessly integrates with `ImageIO.read()`.

== Quick Start

[source,java]
----
BufferedImage image = ImageIO.read(new File("image.wmf"));
----

== Maven Coordinates

[source,xml]
----
<dependency>
    <groupId>pro.verron.imageio</groupId>
    <artifactId>imageio-wmf</artifactId>
    <version>${project.version}</version>
</dependency>
----
