= ImageIO Modules

This project provides a collection of ImageIO plugins for various vector and metafile formats.
It allows Java applications to read formats like SVG, EMF, and WMF using the standard `javax.imageio` API.

== Modules

* link:imageio-svg/README.adoc[imageio-svg]: Support for Scalable Vector Graphics (SVG).
* link:imageio-emf/README.adoc[imageio-emf]: Support for Enhanced Metafile (EMF).
* link:imageio-wmf/README.adoc[imageio-wmf]: Support for Windows Metafile (WMF).

== Key Features

* **Standard API**: Integrated with `ImageIO`, no need for custom loading logic.
* **Vector Support**: High-quality rendering of vector formats to `BufferedImage`.
* **Lightweight**: Optimized for performance and minimal memory footprint.
